Dieses Repoistorium enthält den Code für die in der Dissertation "Künstliche Intelligenz im Gießereiwesen Stufenmodell und Einführungsmethode für Anwendungen in der Qualitätssicherung" vorgestellte Anwendung zur Gruppierung von CT-Daten von Aluminium-Gussbauteilen. Die Daten sind auf Anfrage vom Author zu erhalten.
