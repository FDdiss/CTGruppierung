Dieses Repoistorium enthält den Code für die in der Dissertation "Stufenmodell mit iterativer Implementierungsmethode für KI-Anwendungen in der Qualitätssicherung des Gießereiwesens" vorgestellte Anwendung zur Gruppierung von CT-Daten von Aluminium-Gussbauteilen. Die Daten sind auf Anfrage vom Author zu erhalten.
